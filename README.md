# Rating System

Система управления рейтингами студентов. Написана на JavaFX с использованием PostgreSQL.

## Быстрый старт

### 1. Требования
* Java 21
* Maven
* Docker (для запуска базы данных)

### 2. Настройка базы данных
Запустите базу данных через Docker Compose:
```bash
export DB_PASSWORD="ваш_секретный_пароль"
docker-compose up -d
```
База данных будет доступна на порту `54331`.

### 3. Настройка приложения
Приложение ищет файл `application.properties` в следующих местах (по приоритету):
1. Системная папка (`/etc/rating-system/` или `C:\ProgramData\RatingSystem\`)
2. Текущая директория (рядом с исполняемым файлом)
3. Внутри JAR-архива (настройки по умолчанию)

Вы можете переопределить настройки через переменные окружения:
* `DB_HOST` (по умолчанию: localhost)
* `DB_PORT` (по умолчанию: 54331)
* `DB_NAME` (по умолчанию: rating_system)
* `DB_USER` (по умолчанию: rating_user)
* `DB_PASSWORD` (ОБЯЗАТЕЛЬНО)

### 4. Сборка и запуск
Если вы скачали исходный код:
```bash
./run build
./run app
```

Если у вас есть готовый JAR-файл:
```bash
java -jar rating-system.jar
```
*(Убедитесь, что рядом с JAR лежит файл `application.properties` или установлена переменная `DB_PASSWORD`)*

## Безопасность
* Пароли пользователей хешируются с помощью BCrypt.
* Подключение к БД требует обязательной установки пароля через переменную окружения `DB_PASSWORD`.
* Регистрация новых пользователей по умолчанию дает роль `GUEST` (только просмотр). Для повышения прав обратитесь к администратору БД.

## Экспорт и импорт
Приложение поддерживает экспорт отчетов в PDF и обратный импорт данных из этих PDF-файлов.
