# Rating System

Система управления рейтингами студентов. Написана на JavaFX с использованием PostgreSQL.

## Быстрый старт

### 1. Требования
* Java 21
* Maven
* Docker (для запуска базы данных)

### 2. Настройка базы данных
Приложение настроено на работу с удаленным сервером **автоматически**. Вам **не нужно** ничего настраивать или запускать локально для базовой проверки.

Если вы хотите запустить свою локальную базу данных:
1. Установите Docker.
2. Выполните `docker-compose up -d`.
3. Измените `db.postgresql.host=localhost` в файле `application.properties`.

### 3. Настройка приложения
Приложение работает "из коробки". Оно автоматически подключается к центральной базе данных, что позволяет пользователям сразу видеть общие данные.

### 4. Сборка и запуск
Если вы скачали исходный код:
```bash
./run build
./run app
```

Если у вас есть готовый JAR-файл:
```bash
java -jar rating-system.jar
```
*(Убедитесь, что рядом с JAR лежит файл `application.properties` или установлена переменная `DB_PASSWORD`)*

## Безопасность
* **Защита БД:** Приложение использует ограниченного пользователя `rating_app_user`. Он может добавлять и изменять данные, но у него **нет прав на удаление** записей или таблиц. Это защищает общую базу от вандализма.
* **Пароли:** Пароли пользователей хешируются с помощью BCrypt.
* **Роли:** Регистрация новых пользователей по умолчанию дает роль `GUEST` (только просмотр). Повышение прав до администратора выполняется напрямую в БД.

## Экспорт и импорт
Приложение поддерживает экспорт отчетов в PDF и обратный импорт данных из этих PDF-файлов.
